# 報名與初選

在各方宣傳之下，短短一個星期內，RAY計畫湧入近百位報名者。

除了性別比例和地區均衡，
PDIS 小組初選時也同時考量了不同專長，希望讓見習生的組成更加多元；
履歷是否完整，也是PDIS小組的重要參考依據。
