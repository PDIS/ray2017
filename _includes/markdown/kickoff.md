# 籌備會議

本案源於本(106)年5月間線上報稅系統因無法相容Safari瀏覽器，引發民眾抱怨，其後銓敘部年金試算系統亦有小瑕疵阻礙Mac使用者近用， 嗣PDIS小組針對報稅軟體導入開放政府程序，與財政部及利害關係人協作改善，亦及時提供技術建議解決年金試算系統問題，奉示應就政府機關網站是否有類似問題研議解決方案，以達成跨系統、平台及裝置平等進用的目標，
